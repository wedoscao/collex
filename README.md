---

## Introduction

**Collex** is a lightweight, modular, and extensible collection library written in pure C.  
It provides common data structures with consistent interfaces.

---

## How to Use

### Build the library

```sh
make build
```

### Link to your project
```sh
clang -Iinclude -Lbuild/lib -lcollex your_program.c -o your_program
```
